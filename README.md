### Como rodar
**Djikstra**

Navegue pelo terminal até a pasta do projeto. No caminho `/teoriadosgrafos/djikstra`, execute os seguintes comandos:

```bash
make
./djikstra -h
```

A partir do guia de ajuda, teste o algoritmo adicionando os parâmetros desejados.
